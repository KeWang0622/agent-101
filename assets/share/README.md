## Suggested tweet patterns

**Standalone hook** — pair `the-loop-poster.png` with one line:
> the entire agent loop is six lines.
> every coding agent on Earth wraps these.
> [poster image attached]

**The framework take** — pair `vs-frameworks.png` with:
> "what should I learn for agents — LangChain or LangGraph or…"
> if you want to *understand* agents (vs use them), this is what each option gives you:
> [vs-frameworks image attached]

**Mid-thread reinforcement** — drop a motto card on tweets 3-5 of a launch thread. They keep the visual rhythm and give the thread re-shareable atomic units.
