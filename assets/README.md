# Visual assets

## launch.gif

Generated by `vhs`:

```bash
brew install vhs
export ANTHROPIC_API_KEY=sk-ant-...
mkdir -p src && touch src/main.py
echo "def main(): pass" > src/main.py
vhs assets/launch.tape
# → assets/launch.gif
```

The tape uses `Wait+Line` patterns to track the real agent's output rather than fixed sleeps, so the GIF stays accurate even if model latency drifts.

## og.png — social preview (1280x640)

Build in Figma (~10 min):

1. New file → Frame 1280×640 → fill `#1e1e2e` (Catppuccin Mocha base).
2. Top text "agent-101" at (40, 32), 32px JetBrains Mono Bold, color `#89b4fa` (blue).
3. Two 1192px horizontal lines at y=96 and y=512, stroke `#313244`.
4. Headline "Build a Claude-Code-shaped agent harness from scratch." at (64, 144) — 88px JetBrains Mono Bold, color `#cdd6f4`, three lines, 1.0 line height.
5. Subhead "18 chapters · ~2,000 lines · 3 providers · 36 tests" at (64, 432), 28px JetBrains Mono Medium, color `#a6adc8`.
6. Chapter ladder at (64, 552) — `01 02 03 04 05* 06 07 08 09 10* 11 12 13* 14 15 16 17 ★` — 24px JetBrains Mono Regular, hero markers (`05*`, `10*`, `13*`, `★`) recolored to `#cba6f7` (mauve) and bolded.
7. Export 2x PNG → `assets/og.png` → upload via repo Settings → Social preview.

Lock it. Never touch it again.
