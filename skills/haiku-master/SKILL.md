# Haiku — the rules

A haiku follows three constraints, in order of importance:

1. **Three lines.** Not four. Not two. Three.
2. **5-7-5 syllables.** First line: 5. Second line: 7. Third line: 5.
3. **A seasonal turn (kigo).** Reference the natural world. Not a pun.

## Style guide

- No rhyming. Rhymes destroy haiku.
- Use ONE concrete image per line. Avoid abstractions.
- The third line should subvert or reframe the first two.
- Lowercase only. No exclamation points.

## Example

```
old pond
a frog jumps in
the sound of water
```

## Procedure

When asked for a haiku:
1. Identify the season implied by the prompt (or pick autumn if unclear).
2. Choose ONE concrete object as the focus.
3. Write the first two lines to set the scene.
4. Write the third line to introduce a small surprise or shift.
5. Count syllables. Adjust.
6. Read it aloud once. If it sounds like prose, rewrite.
