agent-101

A Claude-Code-shaped agent harness, built chapter by chapter.

agent.py 

      

    


    

      
What am I?

      

        I am agent-101 — an educational reference implementation of an
        AI coding agent. I wrap the Anthropic Messages API
        in a tool-use loop, exposing every primitive in plain, readable Python.
      

      

        Think of me as the engine behind
        Claude Code, stripped down to ~840 lines so you can read every
        decision on a single flight.
      

    


    

      
What I can do

      

        

          
🔧

          
7 Built-in Tools

          
Read · Write · Edit · Bash · Glob · Grep · TodoWrite

        

        

          
🔄

          
Tool-Use Loop

          
Claude requests tools; I run them; results feed back — until done.

        

        

          
⚡

          
Streaming Output

          
SSE chunk-buffered streaming so you see tokens as they arrive.

        

        

          
💾

          
Session Persistence

          
JSONL sessions, resumable with --resume <id>.

        

        

          
🗜️

          
Context Compaction

          
Auto-compacts at 60 % of the model window to keep long sessions alive.

        

        

          
💰

          
Prompt Caching

          
5-minute TTL cache on system + tools — ~10× cost reduction.

        

        

          
🔌

          
MCP Support

          
Speaks the Model Context Protocol over stdio (chapters 13 & 14).

        

        

          
🛡️

          
Permission Modes

          
ask · allow · deny — per tool call or globally.
